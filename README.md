# Yams Scoreboard

Tabel de scor digital pentru jocul Yams (Yahtzee), sub formă de PWA cu comandă vocală în română.

Suport 1-4 jucători, zaruri 2D/3D cu sunete, salvare locală, istoric, statistici și grafice.

🔗 Live: https://munteancd.github.io/yams-scoreboard/

> Comenzile vocale necesită Chrome/Edge pe Android (Web Speech API) și HTTPS.
